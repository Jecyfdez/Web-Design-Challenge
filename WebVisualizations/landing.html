SUMMARY: LATITUDE VS. X

The purpose of this project was to visualize the weather of over 500 cities across the world to see how weather changes as we get closer to the equator. We utilized a simple Python library, the Open Weather Map API and some creativity to visualize the weather across the world.

This site provides the data we used to observe the weather. All the visualizations created from the dataset include a description and explanation of the trends and correlations found.
